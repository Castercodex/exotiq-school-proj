--HOW TO RUN THE CODE--
This is an Express backend server hosting a rest api.

in root folder run npm install
then run npm start

--API ENDPOINTS--
GET /api/cars
GET /api/cars/:id
POST /api/cars
/ HOME PAGE
/admin/listing ADMIN PAGE

database is in the root folder as data.sql

--TECHNOLOGIES USED--
Node.js
Express
Mysql
HTML
CSS
Vanilla Javascript
